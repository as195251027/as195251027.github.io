# css

- 含义
    - 样式表（层叠样式表
    - 引入方式:
        - link标签 ---外联---(推荐)
        - style标签(在头部（head）书写) --内嵌(做测试)
        - style属性:style属性值就是css代码  ----内联

- 书写规范

    - 选择器 你要为页面中的哪一块设置样式

    
    - 基础选择器的优先级
        - id>class>tag>*


```css
*{
    属性名: 属性值
}

.div{
    属性名: 属性值
}

```
## 常见属性整理

- 文本属性

- 边框属性

- 背景属性

- 盒子模型属性
    - 内边距
    - 外边距
    - display属性 

- 布局属性
    - 表格布局
    - 定位布局
        - position属性
    - 浮动布局
        float属性
    - 弹性盒子布局
        - display:flex

- 基础选择器
    - '*'通配符选择器
    - 'tagName' 标签选择器 直接使用标签名 
    - '.className' class选择器，给标签添加一个class属性
    - '#idName' id选择器。同一个id名在同一个页面只能出现一次。
